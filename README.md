# Metareasoning CARINA

## Requirements

- Python 3
- [virtualenv](https://pypi.org/project/virtualenv/)

## Create virtualenv

Go into the project's root folder and type the command below

```bash
virtualenv .venv
```

## Activate virtualenv

```bash
source .venv/bin/activate
```

## Install dependencies

```bash
pip install -r requiremets.txt
```

## Run

```python
python main.py
```
